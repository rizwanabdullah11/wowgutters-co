# OG Images — Action Required

These images are used as Open Graph previews when pages are shared on social media (Facebook, WhatsApp, LinkedIn).

## Required images (1200×630px JPG)

| File | Used on |
|------|---------|
| `default.jpg` | Homepage and all pages without a specific OG image |
| `birmingham.jpg` | Gutter Cleaning Birmingham page |
| `solihull.jpg` | Gutter Cleaning Solihull page |
| `wolverhampton.jpg` | Gutter Cleaning Wolverhampton page |
| `coventry.jpg` | Gutter Cleaning Coventry page |
| `walsall.jpg` | Gutter Cleaning Walsall page |
| `dudley.jpg` | Gutter Cleaning Dudley page |

## What to use

Use a before/after gutter cleaning photo at 1200×630px saved as JPG.
- Before: blocked gutter full of leaves/moss
- After: clean gutter with clear water flow

Or use a photo of a WOW Gutters technician at work with the company van visible.

## Fallback behaviour

If a specific OG image file doesn't exist, Next.js will fall back to the default.jpg.
Until real photos are added, copy your best gutter cleaning photo to `default.jpg`.
